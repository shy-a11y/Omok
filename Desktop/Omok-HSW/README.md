# Omok
자바 쓰레드와 소켓통신을 이용해 GUI를 통한 오목 게임 프로그램 제작 프로젝트

Thread, Socket, Stream, GUI을 주로 사용했으며 Header 프로토콜을 만들어 데이터 통신이 확인하기 쉽도록 만들었습니다.

[시연영상 (유튜브) : 1분 55초 입니다.](https://www.youtube.com/watch?v=7EFXMMnH98s)
